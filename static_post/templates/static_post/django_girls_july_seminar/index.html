HTML과 CSS의 기본과 활용

HTML과 CSS로 HTML과 CSS를 보면서 배워봅시다.

안녕하세요. 7월 장고걸스 세미나를 하게 된 최현묵 입니다.

HTML

HTML은 무엇인가요?

HTML은 HyperText Markup Language의 약자입니다.

HyperText

HyperText는 한 문서에서 다른 문서로 이동이 가능한 텍스트를 의미합니다.

Markup Language

Markup Language는 문서나 데이터의 구조를 명기하는 언어입니다.

HTML의 기본 구조

doctype

<!doctype html>
위와 같은 형태로 최상단에 선언 되는 doctype은 여러가지 html 버전이 있기 때문에, 어떤 버전의 HTML을 사용할 것인지 명시해줍니다.

html

<html> ... </html>
와 같은 형태로 문서를 감싸줍니다.

head

<head> ... </head>
html 태그와 마찬가지로 이런형식으로 감싸집니다. 기본적으로 html의 대부분 태그는 이런식으로 감싸진다는 걸 기억하세요. (예외도 존재합니다.) head 태그는 보통 컴퓨터에게 알려주는 정보들을 명기합니다.

meta

meta 태그는 작성되고 있는 HTML 문서 자체의 정보들을 명기합니다.

style

style 태그 안에는 보통 css와 같은 스타일시트의 내용이 들어옵니다. 


<style type="text/css">

/* 주석은 요렇게! */

body {

    background-color: rgba(155, 155, 0, 0.2);

}

</style>

link

link 태그는 외부 자원을 연결해주는 데 사용됩니다. 스타일시트를 링크하는데 많이 사용합니다.

<link rel="stylesheet" href="/css/djangogirls.css">

script

스크립트 코드를 적기 위해 사용합니다. 


<script>
    alert("안녕하세요! 만나서 반갑습니다.");
</script>

보통은 아래와 같이 외부의 코드를 불러와서 사용합니다. 

<script src="https://code.jquery.com/jquery-3.1.0.slim.min.js" ></script>

body

보통 실제 유저에게 보여지는 정보들을 적어줍니다. 지금 여러분이 보시고 있는 화면도 body 태그의 내용이에요!

div

div 태그는 가장 많이 쓰이는 태그 중 하나입니다. division의 약자이고, html 내에서 블록(?)을 구분할 때 쓰입니다.

span

span 태그는 inline 블록 태그입니다. div의 경우 같은 라인이 바로 구분되지만 한 라인에서 블록을 구분 할 수 있습니다. inline 태그라는 속성을 가지고 있기 때문에, span 태그 안에 div 태그를 집어넣을 수 없습니다. 물론 반대의 경우는 가능합니다.

heading(h1 ... h6)

heading 태그는 보통 제목을 표시할 때 많이 사용합니다. <h1>와 같은 형태로 표현하며, 아래와 같이 6가지로 표현됩니다.

안녕하세요. 저는 h5입니다.
안녕하세요. 저는 h6입니다.

p

p 태그는 paragraph의 약자로 문단 또는 단락을 나눌 때 사용됩니다.

a 태그

a 태그는 anchor의 약자로 보통 다른 문서 또는 위치와 연결(link)할 때 많이 사용됩니다.

<a href="http://www.djangogirlsseoul.org/meltingpot-seminar-july/">django girls 7월 세미나</a>

위 코드는 아래와 같이 보여집니다.

django girls 7월 세미나

ul, ol, li

위 태그들은 리스트를 표현할 때 사용합니다. li 태그는 리스트의 아이템 요소 입니다.

<ul>
<li>첫번째 입니다.</li>
<li>두번째 입니다.</li>
</ul>

ul(unordered list)

순서가 없는 리스트로 아래와 같이 표현됩니다.

  • 첫번째 입니다.
  • 두번째 입니다.
  • 세번째 입니다.
  • 네번째 입니다.

ol(ordered list)

순서가 없는 리스트로 아래와 같이 표현됩니다.

  1. 첫번째 입니다.
  2. 두번째 입니다.
  3. 세번째 입니다.
  4. 네번째 입니다.

빈 태그(empty tag)

내용이 필요없는 태그들은 빈 태그라고 불립니다.

br

                  강제로 줄바꿈을<br>할수 있는<br>태그입니다.

강제로 줄바꿈을
할수 있는
태그입니다. 하지만 권장되지 않는 태그로서 사용을 자제 하는 것이 좋습니다.

img

img 태그는 이미지를 표현하는데 사용합니다. src 속성에 주소(url)를 입력하면 됩니다.

<img src="img_src.png" alt="본 발표자료의 첫 화면입니다." />

input

input 태그는 사용자의 입력을 받을 수 있는 태그입니다.

<input type="text" name="name" value="">

그 외의 태그들

button, table 관련(th, thead, td, tbody 등), 리스트 (dl, dt, dd), section, header, footer, code 등등

제가 말씀드린 태그 외에도 굉장히 많은 태그들이 존재합니다.

W3C

http://www.w3.org

W3C(World Wide Web Consortium)는 월드 와이드 웹을 위한 표준을 개발하고 장려하는 조직으로 팀 버너스 리를 중심으로 1994년 10월에 설립되었다. W3C는 회원기구, 정직원, 공공기관이 협력하여 웹 표준을 개발하는 국제 컨소시엄이다. W3C의 설립취지는 웹의 지속적인 성장을 도모하는 프로토콜과 가이드라인을 개발하여 월드 와이드 웹의 모든 잠재력을 이끌어 내는 것이다.
